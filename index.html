Recent Work

Kiddos

Kiddos is an online shopping clone for kids' clothing, which allows users to do shopping on an eCommerce website through picture feeds. The user can explore children's clothes and save their favorite items in their favorite list, or in the cart. Created with React/Redux in the frontend and Flask in the backend.

OverShare

OverShare is a reactive Single-Page Instagram-like app that allows users to view, like, and comment on photos. Users can search for other users by a weighted search algorithm to build AJAX autocomplete suggestions. OverShare reduced the server load by utilizing AWS S3 to implement the user photo upload.

  • Repo

ShowMe

ShowMe is a Goodreads clone for TV shows where users can find, rate, and comment on television shows, as well as save them to their self-created ShowShelves. ShowMe used Bcryptjs algorithms to hash the password that was stored in the database. Express was used to create the server routes to get, post, delete, or patch data from the database.
